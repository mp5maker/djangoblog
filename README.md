**Installation**
1. Pip
2. Python v36-32
3. Node/Npm
4. virtualenv, virtualenvwrapper

```bash
mkdir src
mkvirtualenv djangoblog
touch requirements.txt
pip install freeze
```

requirements.txt
```txt
Django==1.11
djangorestframework==3.7.7
```

```bash
pip install -r requirements.txt
cd src
django-admin startproject settings .
python maange.py startapp pages
```

**Requirements**
1. Django 1.11
